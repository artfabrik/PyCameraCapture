PyCameraCapture
=========

> A basic python wrapper of GPhoto2.

Requirements
-----------
  - python2.7
  - pyside
  - pyside-tools
  - optional
    - qt-sdk
    - qt-designer (on linux:/usr/share/qt4/bin/designer)
    - qt-creator


Getting Started
--------------
Test if gphoto works correctly
```
gphoto2 --auto-detect
```

run with
```
python capture.py
```


Gui Creation
--------------
make .ui file with qt-designer
convert to .py file with pyside-uic
